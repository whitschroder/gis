# Lab 7: Hydrology

In this lab, students will conduct a hydrological analysis.

The data for this week's lab includes 4 ASTER DEM tiles:

1. ASTGTMV003_N14W091_dem.tif
2. ASTGTMV003_N14W092_dem.tif
3. ASTGTMV003_N15W091_dem.tif
4. ASTGTMV003_N15W092_dem.tif

For this lab, combine these tiles into a single DEM and run the hydrological analysis tools discussed in the following sections. Describe your workflow and produce the following:

1. A map showing a flow accumulation with your defined threshold (with lines thickened, either by converting to vector or running Focal Statistics/Aggregate)
2. A map showing stream order
3. A map showing basins
4. A map showing watersheds
